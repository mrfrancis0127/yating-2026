字體組合：**Noto Serif TC**（中文標題）+ **Cormorant Garamond**（英文裝飾）+ **Bebas Neue**（數字）

## 🌐 部署到 GitHub Pages

1. 在 GitHub 建立新 repo，例如 `fang-yi-2026`
2. 將整個 `website/` 資料夾的內容（不含資料夾本身）上傳到 repo 根目錄
3. 進入 repo → Settings → Pages
4. Source 選 `Deploy from a branch`，Branch 選 `main` / `/(root)`
5. 等 1～2 分鐘，網站會出現在 `https://你的帳號.github.io/fang-yi-2026/`

## 🛠 修改重點提醒

- 文字內容直接在 `index.html` 中修改
- 圖片替換時，保持相同檔名最方便（不必改 HTML）
- 聯絡資訊在頁面底部 `.contact-grid` 區塊
- 顏色集中在 CSS 開頭 `:root` 區塊（CSS 變數），改一次全站套用
